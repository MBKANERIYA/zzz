# TaxFlow Accounting Services — Website

## Overview
Premium HTML landing page for TaxFlow Accounting Services, a Sydney-based registered tax agent and CPA-led accounting firm.

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter)
- No frameworks — static page

## Key Design Decisions
- Corporate blue palette (`#2563eb` primary)
- Inter font family
- CSS custom properties for theming
- IntersectionObserver for scroll animations
- FAQ accordion with structured data (JSON-LD)
- Fully responsive (mobile-first breakpoints at 768px and 1024px)

## SEO
- FAQ Schema markup included
- Semantic HTML5 structure
- Meta title and description set
- Proper heading hierarchy (single H1)
